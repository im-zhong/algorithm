# Backtrack

1. 抽象地说，解决一个回溯问题，实际上就是遍历一棵决策树的过程，树的每个叶子节点存放着一个合法答案。你把整棵树遍历一遍，把叶子节点上的答案都收集起来，就能得到所有的合法答案。
2. 站在决策树的每一个节点上（也就是一个节点的前序位置），你现在就在做决策，你可以选择的选项，就是选择列表，必须全排列 2 已经有一个2了，那么下一个数字只能选1和3，如果选了1， 21， 你的选择列表就只有3了！！！也就是你的选择会修改选择列表！所以在前序位置，做出选择，相应的修改选择列表；在后序位置，撤销选择，相应的也要恢复选择列表
3. 现在可以解答开头的几个名词：[2] 就是「路径」，记录你已经做过的选择；[1,3] 就是「选择列表」，表示你当前可以做出的选择；「结束条件」就是遍历到树的底层叶子节点，这里也就是选择列表为空的时候。
